# Problem Solving Repository 📝

Each folder focuses on a specific type of problem to help organize my learning and track progress.

---

## 📂 Folders and Content

- **2d_array_problem** – Problems and solutions using 2D arrays.  
- **Array_problem** – General array practice problems.  
- **condition_problem** – Problems using conditional statements (if/else/switch).  
- **function_problem** – Practice problems using functions.  
- **loop_problem** – Problems focusing on loops (for, while, do-while).  
- **nested_problem** – Problems with nested loops or conditions.  
- **pointer_problem** – Practice problems using pointers.  
- **string_problem** – Problems related to string manipulation.  
- **structure_problem** – Practice problems using structures.
